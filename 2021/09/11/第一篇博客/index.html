一、环境
1、首先按顺序安装git和node.js
2、在桌面打开git输入git –version验证git是否安装成功，再输入node -v验证node.js是否安装成功
3、新建存放博客的文件夹，右键打开git，进行安装hexo，输入安装命令npm install hexo -g
4、输入命令hexo -v测试是否安装成功
5、输入命令npm install –save hexo-deployer-git安装hexo依赖
二、git配置SSH key
1、先注册一个github账号（为什么要配置SSH key，因为可以免密的将本地的源码和资源上传到github）
2、打开桌面打开git，输入命令cd ~/.ssh看本地是否配置好SSH key
3、配置ssh，输入生成命令ssh-keygen -t rsa -C “邮件地址”，连续按三次回车
4、可发现在电脑c盘用户文件夹出现.ssh文件夹
5、打开.ssh文件夹，再打开id_rsa.pub（注：该文件需要用记事本打开），将全部内容复制下来
6、打开github主页，点击个人设置，点击左侧的SSH and GPG keys，点击New SSH key
7，将id_rsa.pub复制的内容粘贴到key中，title随便起一个，点击 Add SSH key
8、输入ssh -T git@github.com测试是否成功，接着输入yes，出现Hi 用户名！说明成功
9、配置账号密码$ git config –global user.name “用户名”
$ git config –global user.email “邮箱”
三、搭建博客
1、在存放博客文件夹打开git，输入命令hexo init初始化个人博客，出现start blogging with Hexo！即成功
2、打开存放博客的文件夹可以发现出现了许多文件夹
3、在存放博客文件夹打开git，输入hexo g生成静态网页
4、继续输入hexo s进行预览
5、接着打开github新建一个仓库，注意仓库名称格式为用户名.github.io，下滑点击Add a README file，点击创造仓库
6、打开存放博客的文件夹，编辑_config.yml文件，下滑最底部输入
deploy:
type: git
repository: git@github.com:wind3045/wind3045.github.io.git
branch: main
7、在存放博客文件夹打开git，输入hexo d进行发布到github
8、最后就登入https://wind3045.guithub.io即可看到自己的博客
